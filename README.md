# Learn-cpp
My c++ learning journey - Games and DSA

## Projects

### 1. Tic Tac Toe Game
- **Features**: 2 Player Console Game, Win/Draw Detection, Input Validation
- **Concepts**: 2D Arrays, Functions, Loops, Conditional Logic
- **File**: `tic_tac_toe.cpp`

## How to Run
1. Clone this repo
2. Compile: `g++ tic_tac_toe.cpp -o game`
3. Run: game 
4. Players enter 1-9 to mark their move
